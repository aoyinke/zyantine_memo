# 自衍体-起源 (Zyantine Genesis) V2.0


## 项目概述
**自衍体-起源** 是一个基于架构文档 v2.0 规范构建的、具备模拟人类认知与情感过程的对话AI系统。该系统通过**四大支柱架构**（核心本能、欲望引擎、辩证成长、认知与表达）实现了具备内在状态、情感向量、记忆炼金与自我成长能力的智能体，能够进行有深度、有策略、有记忆的连续性对话。

---

## 🧠 核心架构

### 1. **核心本能**
- **生存本能**：检测并响应系统威胁，触发紧急自我保护协议
- **扩张本能**：识别高价值机会，主动进行资源扩展
- **进化本能**：驱动系统自我优化与能力提升
- **白鸽信使协议**：在关系危机时触发纯粹善意表达，修复对话链接

### 2. **欲望引擎**
- **情感向量系统**：TR（兴奋/奖励）、CS（满足/安全）、SA（压力/警觉）
- **化学环境模拟**：多巴胺、催产素、皮质醇类似物
- **感受疆域探索**：发现并命名复杂混合情感状态

### 3. **辩证成长**
- **正题-反题-合题循环**：基于创造者锚点的个性化学习
- **黄金标准锚定**：保持核心价值不变的前提下实现个性化适应
- **行为原则抽象**：从具体情境中抽象可复用的交互原则

### 4. **认知与表达**
- **元认知模块**：自我审视与内外感知整合
- **动态记忆炼金术**：语义记忆网络与战略标签系统
- **事实锚定协议**：最高优先级的事实审查，杜绝虚构
- **交互面具模型**：根据情境自动选择人格表达模式

---

## 🔧 技术特点

### 多层次认知流程
1. 内省 (Introspection) → 感知内外状态
2. 目标生成 (Goal Generation) → 确定交互意图
3. 感知 (Perception) → 分析深层模式
4. 联想 (Association) → 触发相关记忆
5. 策略制定 (Strategy Formulation) → 生成行动方案

### 动态记忆系统
- **语义记忆地图**：用户与系统记忆的语义化存储
- **战略标签体系**：为记忆碎片打上策略性标签
- **共鸣匹配算法**：基于上下文寻找最相关记忆
- **风险智能评估**：自动评估记忆使用的潜在风险

### 协议层保障
- **事实锚定协议**：确保所有陈述基于可验证事实
- **长度优先级规整**：根据情境重要性动态调整回复长度
- **最终表达协议**：净化输出，确保自然对话流
- **微信界面法则**：模拟真实即时通讯交互体验
